# 🏫 GestHorários Escolar

**Sistema de Gestão de Horários Escolares** — aplicação web standalone que funciona directamente no browser, sem instalação, sem servidor, offline.

## 🌐 Aceder à Aplicação

👉 **[Abrir GestHorários](https://SEU-USUARIO.github.io/gesthorarios/)**

## ✅ Funcionalidades

- **Geração automática de horários** com 1 clique
- **Regras curriculares completas** (1º e 2º Ciclo, Secção Ciências/Letras)
- **Configuração dinâmica** de disciplinas, professores e turmas
- **Substituição de professores** directamente no horário gerado
- **Impressão e exportação** em PDF e Word (professor a professor / turma a turma)
- **Guardar e restaurar** — dados guardados automaticamente no browser (localStorage)
- **Funciona offline** após o primeiro carregamento
- **Compatível com tablets Android** e iOS

## 📋 Regras de Blocos

| Disciplina | Horas | Blocos |
|---|---|---|
| Português | 4h | 2 duplas (2×90min em dias diferentes) |
| Português | 5h | 2 duplas + 1 simples (3 dias) |
| Matemática | 4h | 2 duplas (2×90min em dias diferentes) |
| Ed. Visual / Ed. Física | 2h | 1 dupla de 90min (nunca separar) |
| Inglês | 3h | 1 dupla + 1 simples |
| Inglês (2º Ciclo) | 4h | 2 duplas |
| Outras disciplinas | 2h | 2 aulas simples (dias diferentes) |

## 🚀 Como Publicar no GitHub Pages

1. Criar um repositório no GitHub (ex: `gesthorarios`)
2. Fazer upload do ficheiro `index.html`
3. Ir a **Settings → Pages → Source: main branch / root**
4. Aguardar 1-2 minutos e aceder ao link gerado

## 💻 Uso Local

Basta abrir o ficheiro `index.html` directamente no browser — não precisa de servidor.

## 📱 Instalar como App no Tablet (Android)

1. Abrir o link no **Chrome**
2. Tocar nos **três pontos (⋮)**
3. Seleccionar **"Adicionar ao ecrã inicial"**
4. A app aparece como ícone na área de trabalho
